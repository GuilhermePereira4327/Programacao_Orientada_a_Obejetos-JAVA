# Programação Orientada a Objetos com Java

Repositório principal dos meus estudos de **Java + POO** do curso completo do professor **Nelio Alves** (Udemy). 

Aqui estão registrados todos os exercícios, desafios e projetos desenvolvidos durante o curso.

## Objetivo do Repositório

Consolidar os conceitos de Programação Orientada a Objetos aplicados na prática com Java, preparando-me para projetos reais com Spring Boot.

## Tecnologias e Conceitos Estudados

- **Java** (JDK 17/21)
- **Paradigma Orientado a Objetos**:
  - Encapsulamento
  - Herança
  - Polimorfismo
  - Classes Abstratas
  - Interfaces
- Enumerações
- Composição
- Tratamento de Exceções
- Generics
- Collections Framework
- Trabalhando com Arquivos
- UML (básico)
